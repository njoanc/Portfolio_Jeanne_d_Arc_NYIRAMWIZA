Circular Plastic Economy Innovation Hub

The Circular Plastic Economy Innovation Hub, on which I contributed as a Backend Engineer, investigates how universities can drive a thriving ecosystem to support and develop waste to wealth ventures.

This project, powered by TypeScript-Express, TypeORM, MySQL, MongoDB, Socket.io, AWS S3, nodemailer, mocha, chai, and redis, evaluates how key players can combine effort to build capacity and strengthen the possibility of innovations happening in society. Such synergies bring about technical efficiency which is an opportunity to leverage digital transformation to leapfrog some of the most critical sectors of the circular economy in Africa.

Building a culture of innovation in universities is paramount for these possibilities and a budding ecosystem provides an encouraging environment for innovation to occur. An enabling eco-system will help create, transform, and communicate knowledge, therefore improving student employability and nurturing innovations for converting waste into useful products.

This initiative represents a significant step towards a sustainable and innovative approach to waste management, fostering collaboration between academia, industry, and technology to create a circular economy that benefits both society and the environment.

My Contribution as Backend Engineer:

  • Developed backend functionality using TypeScript-Express
  • Utilized TypeORM for database interactions with MySQL and MongoDB
  • Implemented real-time features with Socket.io
  • Integrated AWS S3 for storage needs
  • Incorporated nodemailer for email notifications
  • Ensured robust testing using mocha and chai
  • Utilized Redis for caching and optimization
